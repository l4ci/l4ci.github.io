Notes

Github recently announcend, that they now support PSD image viewing and diffing. It wasn´t possible to see and track changes to .psd files on github - but not it is. Awesome.

Source: https://github.com/blog/1845-psd-viewing-diffing

Google (I wont link to google for obvious reasons: if you dont know what google is or where to find it... google it!) blogged about their recent addition to the google search crawler. It now understands javascript and css. That should change a lot for (so called) SEO experts. They are going to update the Webmaster Tools shortly, so it´s easy for us to get behind that process.

Source: http://googlewebmastercentral.blogspot.de/2014/05/understanding-web-pages-better.html

Links

Links to various interesting / creative / new stuff on the web I came across this month.

  • 101 essential free fonts for web designers - webdesignerdepot.com
    I normally don´t read post that seem that start with "Best 100" or "Top 10" - but this one was quite useful and interesting for anyone using typographie. So basically anyone in the webdesign industrie.

  • Basehold.it - basehold.it
    Baseline grids are super helpful for web designers, allowing us to maintain vertical rhythm on our designs. But short of a few Javascript bookmarklets, there isnt really a simple way to get a baseline grid overlay on our designs. Until now!

  • FitText - fittextjs.com
    FitText makes font-sizes flexible. Use this plugin on your fluid or responsive layout to achieve scalable headlines that fill the width of a parent element.

  • ColorFinder - pieroxy.net/blog/pages/color-finder/
    You need the most prominent color in an image? ColorFinder might come in handy!

  • Underscores - underscores.me
    Underscores makes creating wordpress themes from scratch easy. Well commented, modern, HTML5 templates. 404 templates and much more.

As always I hope you might find some of them useful. Enjoy the sun and take care!
