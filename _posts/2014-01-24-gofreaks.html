When I created the first homepage for the gofreaks.eu clan, it was only meant to be a simple page for presenting it to the outside world. The whole organisation and communication was arranged in facebook: A main facebook group for clannews, organisation, game related talk and member communication.

This worked out for quite some time, but lately they struggeld to keep up with all the new members. The main group exploded and it was very hard to keep up with all the information that got posted. They started to create more groups for all the different games, but that created an invisible wall between the clan members. So something new had to be done...

Building a community

I started to develop a new site. A place, where every member can interact with others and also get all the related information for his games without distraction. A place where you could easily get information, like his steam or esl name, about another member without asking them directly.

Since facebook was always the main plattform for the clan, I integrated the login/registration process directly with facebook. So anyone who was already in the clan (and so in facebook) could easily register on the page without creating a user account manually.

A whole new

Profil Page

Every member has a profil page, where he/she can enter all the related information about him/herself. Things like Steam, Origin, Battle.net account information, but also links to twitch.tv, esl and facebook.

I also wanted to give them the option to make all these (quite sensible) information available to members only. So there is the option to do exactly that. Guest only see the avatar and nickname, when a profil is privat.

” We want to be different. New! “ Roland Otto - Clanleader

Communication

That is one of the keypoints that had the be solved. A forum is always a nice idea, but with all the different games this wasnÂ´t enough. For this reason I added a new functionality to the site, which is quite unique.

There is a list of games on the site, that continuosly grows. Every member can join a game and by doing that some things happen:

  1. The member is listed under the game
  2. The game gets listed on the members profil
  3. The gameleader is notified
  4. The member gets access to the game related forums

The last point is the most important one. With this process it was ensured, that every member only gets the information related to his game. There were some fights about this, because some people got annoyed on how others talk about their game in the old facebook groups.

Forum

So the forum is the main communication tool now, where game related talk is only available to those interested in them and all the other clan related information is available at all times.

Notifications

After testing the site with some members the most asked feature was a notification system. As everyone was used to from facebook, you get a notification when something happens that is related to you.

Therefore I developed the so called INS Instant Notification System. It tracks every activity on the site, and notifies the right members of new updates everytime. It even works without refreshing the page.

A Better mobil experience

Although this step is not 100% finished yet (as the whole site is still in beta), the page is fully responsive. That means it not only looks good on any screen device, its also fully functional. In other words, members can read and post in the forum or read other news on the site, when visiting the page from their mobile device.
